Summary¶

Homepage¶

The home page can be customized to meet the user’s need. You can add widgets to see the different modules within the system as well as the last dashboards and models used. Widget editor

Properties tool¶

The properties bar is located on the left margin of the window. You can use it to access all the modules within the system; however, you have access only to the modules enabled in your profile. The bar contains the following buttons:

|Icon|Description| |–|–| ||Home: It leads to the home page.| ||File manager: It leads to the file manager window, where you find all the models. (Go to file manager)| ||Schedule task: It gives access to the module where you can set up the automatic tasks to run models, OleDB and ETL. After it has been run it can be seen by other users, or just closed.| ||Analysis: It leads to the analysis module. (Go to analysis)| ||Influence diagram: It shows the influence diagram of the model on which you are working. This option is available only when a model is open. (Go to Influence diagram)| ||Reports manager: It gives access to the module that contains all the dashboards and reports created by the user and those that have been shared with the user. This option is available only when a model is open. (Go to reports manager)| ||Workflow: It gives access to the different modules: Gantt, Status, Transitions, Transitions by rol and Task groups. ||Knowledge base: It gives access to the module that contains all the model data. This option is available only when a model is open. (Go to knowledge base)| ||Security options: It leads to the modules: user manager, group manager, role manager, and company manager.| ||Configuration: It leads to the modules: session manager, sanity check, and widget editor.|

The top margin has the following format:

The Cubeplan version number is displayed on the left margin; next to it, you find the following button . It enables/disables the vertical toolbar displayed on the left margin. Then, you find the name of the module you are in and the button to close it; if you close the current module, the page returns to the previous module.
You will find the button on the right margin. It allows you to save any changes introduced in the model. This buttom turns yellow when there are changes to save. If you need to save changes in a new version of the model, you can do so by clicking .

On the right, you can see the model currently open. If you click on it, a window will open showing the properties of the model. This window looks like this:

By clicking you close the model you are working on.
Then, you will see the name of the user who is logged in: . If you click here, you will be able to edit your profile.

At the end, you will find the button to log off. (Go to log off)

Widget editor¶

You can customize the home page by using the widget editor.

First, click on the toolbar to your left and select the option .

To your right you will find the buttons described below, these can be moved around and placed in any position within the window.

You can resume a session by clicking , or close one by clicking . You can also create a new session by clicking . In this case, the widget will not show the new session until you open a model.
