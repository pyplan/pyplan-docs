Create Nodes¶

If you want to create a node you have to enable the diagram edit mode first; the following options will be available:

|Button|Node type|Shortcuts| |–|–|–| ||Variable|Ctrl+2| ||Index|Ctrl+6| ||Module|Ctrl+5| ||Text|Ctrl+9| ||Button|Ctrl+0| ||Decision|Ctrl+1| ||Option|Ctrl+3| ||Objective|Ctrl+4| ||Constant|Ctrl+7| ||Function|Ctrl+8|

You can create a node in two ways:

  1. Click a node and drag it to the influence diagram window.

  2. Use the respective shortcut.

When you click a node, you open the dock where you can edit it.

Click here to learn how to select a node, send a node to the back, bring a node to the front, move a node to a module or delete nodes.

You can also create an alias, input node and output node. You can also choose a style, color and insert an image. When you select more than one node you can specify their alignment and size.

Go here to find the shortcuts for many of these functions.
