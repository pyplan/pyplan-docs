Create Blocks¶

  • Created December 15, 2016

  • Category Create dashboards

You can start editing a dashboard by clicking on it. At the center of the block you will find the following buttons:

|Icon|Description| |:–|:–| ||Table: When you search a node, you will view those which are Variable type. If you click on it, they will be displayed in a table. Click here for a detailed explanation on how to edit tables.| ||Graph: As it is for tables, when you search a node you will view those which are Variable type. If you select a node, it will be shown in a graph. [Click here]|(http://www.cubeplat.com:8081/wiki/en/knowledge-base/column-2/) for detailed information on how to edit all types of graphs. ||Diagram: It creates a diagram-type block.| ||Text editor: It creates a text-type block.| ||Filter: When you search a node, you will view those which are Index and Decision type. If you click it, it will be displayed as a selector. Click here for detailed information on how to edit the selector type block.| ||PivotGrid: As it is for tables, when you search a node you will view those which are Variable type. If you select the node, it will be displayed in a dynamic table. Click here for detailed information on how to edit PivotGrid-type graph.| ||Gis Map: It opens a map. Click here for more information about Gis Map-type block.| ||Magic: When you search a node, you may be able to find any type of node. This button allows you to create button, module, and formnode-types of block. These are not available for the previous buttons.|
