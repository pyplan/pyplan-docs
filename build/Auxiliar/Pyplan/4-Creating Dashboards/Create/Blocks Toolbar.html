Blocks Toolbar¶

When the dashboard is on edit mode, the following buttons are displayed:

|Icon|Description| |–|–| ||It resets the block, leaving it blank. The buttons are displayed so you can start over.| ||It allows you to click and drag a block so you can place it where you want.| ||It splits the block in half, creating a new one under the original.| ||It splits the block in half, creating a new one to the right of the original.| ||It deletes the block.|

If you place the cursor over a block a bubble will appear on the screen, as shown in the picture:

First, you will see the brief description of the node. If you click , you will be sent to the Knowledge Base.

Then, you will see the tasks from the group associated to this node. If you click , you will be sent to Workflow. If you click , you create a new task.
You can edit a task from this window by clicking on its name.

There are two buttons to the right of the block name:

When you click , you can select different time lapses.

When you click , you will find the following functions:

Full screen: it fits the view of the block on the screen.Toogle detail: it shows/hides the index details.Column total: it shows/hides all the columns.Row total: it shows/hides all the rows.Export: go to export a node.**
Table & Graph:** if there is a table, you can create a block with its graph, and vice versa. You can place this block to the right of or under the first one.Knowledge base: it allows you to read, create, or edit the information of the node. Go to knowledge base.Workflow: it gives you access to workflow.View in diagram: it allows you to view the node in the influence diagram.
