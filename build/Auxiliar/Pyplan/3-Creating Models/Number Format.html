Number formatÂ¶

This window allows you to set the number format for the graph. You can specify the digit number and whether the missing digits will be replaced by zeros. You can also specify which symbols will be visible. If you want to add a specific symbol, you should define the position it will take in the graph. The changes introduced will be immediately shown in the graph.
