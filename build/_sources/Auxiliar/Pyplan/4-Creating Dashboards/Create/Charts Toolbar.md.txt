# Charts Toolbar
When you edit a dashboard, a properties window opens. Each type of graph has different options but basically we found the following:

## Chart and Filters

In this section you can change the type of graph and the index position in rows and columns.

## Index Sync

In **Index sync** you can see all the indexes that constitute the variable. You can disable from here the synchronization with other blocks (go to [index synchronization](http://www.cubeplat.com:8081/wiki/en/knowledge-base/index-and-drilldown-drillup-synchronixation/)).

## Advanced Options

[![](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2018/01/Screenshot_1-1.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2018/01/Screenshot_1-1.png)

If the  **Chart zoom** check is enabled, you are able to zoom in the graph by selecting a range with your mouse.

The columns with value 0 (zero) can be hidden by enablind the  **Hide empty data** check.

**View index detail** makes the index details visible under the graph.

**Enable drilldown** allows you to break down a variable by clicking on it.

## Appearance

Appearance tab in the property window allows you to set certain appearance preferences. It looks like this:

[![](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/07/line5.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/07/line5.png)

**Title:** select a color for the title, and whether you want it to be visible.

**Legend:** choose to show or hide the legend of the graph; if it is visible, choose to place in any margin of the graph.

**Labels:** if labels are shown in a bar graph, these can be placed inside or outside the bars.

**Axes:** go to [Axes](http://www.cubeplat.com:8081/wiki/en/knowledge-base/properties-window/#Axes).

**Serie colors:** go to  [Serie colors](http://www.cubeplat.com:8081/wiki/en/knowledge-base/properties-window/#Serie_Colors).

**Background color:** select a background color for the block.

**Time line graph:** available only for Line and Scatter-type graphs. The node must have Time index among its variables. If it is enabled, the dashboard will display a timeline where you can select different segments.

**Show markers:** available only for Line-type graphs. If it is enabled, all points along the line will be visible.  
In Line and Column-type graphs display the following check:  
[![series](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/06/series.png)  
](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/06/series.png)It allows you to overlap the columns or bars in the graph.

### Axes

Axes tab in the property window allows you to set certain axes preferences. It looks like this:

[![ax1](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/ax1.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/ax1.png)

**X Y Labels rotation:** when the cursor is place over the bar, the axes values will rotate.

**Customize axes range:** specify min and max values for the graph.

**Show X Y labels:** show/hide the axes values.

**Show title in x axis/legend:** show/hide the axes title.

### Serie Colors

If you click on Serie color in the property tab the following window will open:

[![serie1](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie1.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie1.png)

It allows choosing between the default color and those created by the user.

Click the button Style Library too see, edit, and create styles. The following window will open:

[![serie2](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie2.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie2.png)

It shows all the styles created by the user.

To edit a style, click  [![edit](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/edit.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/edit.png).

To delete a style, click  [![delete](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/delete.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/delete.png).

To create a new style, click  [![new](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/new.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/new.png), The following window will open:

[![serie3](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie3.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie3.png)

You can edit the style in two ways:

The firs way is one by one, by clicking  [![serie4](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie4.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie4.png). The window displays a field where you can specify a value and specify a color for it [![serie6](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie6.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie6.png). You can keep adding values by clicking Add range.

The second way is to do so automatically. You have to select the index and click  [![serie5](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie5.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie5.png). The colors will be assigned to the values randomly; however, you are able to make changes regarding the color, or delete a style.  
[![serie7](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie7.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/serie7.png)

## Style Library

When you click  [![st6](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st6.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st6.png), you open the following window:

This window shows all the styles that have been created by the user.

To edit a style, select it and click  [![edit](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/edit.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/edit.png).

To delete a style, click  [![delete](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/delete.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/delete.png).

If you click  [![new](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/new.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/new.png), you open the window where you can create a new style:

[![st5](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st5.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st5.png)

First, you must enter the style name. then, to create a new range, click  [![st7](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st7.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st7.png). The new range will be automatically added:  
[![st9](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st9.png)  
](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st9.png)

In it, you must specify the start and end. You can also choose the color it will have in the graph.

To delete a range, click  [![st8](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st8.png)](http://www.cubeplat.com:8081/wiki/wp-content/uploads/2016/04/st8.png) .
